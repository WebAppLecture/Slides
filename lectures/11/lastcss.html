Noch einmal CSS3

  • In CSS3 kann man mehrere Hintergründe einbinden, z.B. über 
    selektor { background: url('hintergrund1.gif') no-repeat 0 0, url('hintergrund2.gif') no-repeat 100% 0; }
  • Als Workaround für ältere Browser empfiehlt sich folgendes 
    selektor { /* FALLBACK für NON-CSS3 Browser ... */ } 
body:last-child selektor { /* Für CSS3 Browser */ }
  • Neben Rahmen über border sind auch sog. Outlines möglich, z.B. 
    selektor {
border: 1px solid #000;
outline: 1px solid #699;
outline-offset: -9px; }
  • In CSS3 kann man nun auch explizit das Box-Model durch das IE(5)-Model ersetzen: 
    selektor { box-sizing: content-box; /* Reguläres Box-Model - *meistens* Standard */ }
selektor { 	box-sizing: border-box; /* Nicht standardisiertes Model - vorsicht Webkit-Textboxen! */ }
