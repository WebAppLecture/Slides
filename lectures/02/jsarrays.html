Arrays

  • Damit sich hinter einer Variablen mehr Objekte verstecken können, muss es sich um ein Array handeln
  • Arrays können auf sehr unterschiedliche Art und Weise initialisiert werden: 
    var leer1 = [];
var leer2 = new Array();
var reserviert = new Array(14);
var belegt1 = new Array('Milch', 'Zucker', 'Mehl');
var belegt2 = ['Milch', 'Mehl', 'Äpfel'];
  • Möglichkeiten zum Hinzufügen eines Elements mit push() (ans Ende) und unshift() (an den Anfang)
  • Mit der Eigenschaft length erhalten wir die Anzahl der Elemente zurück, sort() sortiert das Array
  • Zum Entfernen mit splice() wird der Index und die Anzahl benötigt oder wir löschen das letzte mit pop() 
    var kundenliste = [];//Erstellt ein leeres neues Array mit length=0
kundenliste.push('Herbert Maier');//Ein neuer Eintrag drin
kundenliste.push('Max Hilbert', 'Anton Mustermann');//Zwei weitere hinzugefügt
var i = kundenliste.length;//Variable i hat nun den Wert 3
kundenliste.splice(0, 2);//Entfernt zwei Einträge ab Index 0 - es bleibt 'Anton Mustermann'
var j = kundenliste.length;//Variable j hat nun den Wert 1
