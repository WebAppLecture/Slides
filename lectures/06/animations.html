Animationen ohne JS

  • Effekte wie eben können wir natürlich auch ohne CSS3 in JavaScript erledigen (per CSS ist schneller!)
  • Wie sieht es aber nun aus wenn wir nicht nur Übergänge handhaben wollen sondern echte Animationen?
  • Selbst hier können wir auf JavaScript verzichten - CSS3 Animationen! 
    @-webkit-keyframes beispiel { /* Definition der Keyframes */
	0% { background-color: #F00; } /* Erster Frame am Start */
	20% { background-color: #0F0; } /* Frame bei 20% der Zeit */
	60% { background-color: #00F; } /* Dritter Frame */
	100% { background-color: #FFF; } /* Abschlussframe */
}
#someDiv {
	-webkit-animation-name: beispiel; /* Verweis auf keyframes */
	-webkit-animation-duration: 10s; /* Analog Transitions */
	-webkit-animation-iteration-count: infinite; /* auch 1, ... */
	-webkit-animation-timing-function: linear; /* Analog Trans. */
}
  • Animationen gehen bislang nur auf Safari, Chrome, Firefox (-moz-)
