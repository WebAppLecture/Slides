Mehr Schriftarten!

  • Mit CSS3 haben wir die Möglichkeit eigene Schriften zu definieren und zu verwenden
  • Wir können eigens auf unserem Server gespeicherte Schriften verwenden oder extern gespeicherte
  • Die Syntax hierfür definiert in CSS eine neue Schriftart, z.B. 
    @font-face {
	font-family: 'Gentium'; /* Wird der neue Name! */
	src: local('Gentium'), url('../fonts/Gentium.ttf'); /* Bevorzugt nicht downloaden */
	font-style: italic; /* Optional: Charakterisiert Schriftart z.B. Verwendung nur bei italic*/ }
#Verwendung { font: italic bold 14pt 'Gentium'; }
  • Formate? true-type-font (ttf) geht mittlerweile überall - bevorzugt ist WOFF (Web Open Font Format)
  • Über Google Webfonts1 kann man an Schriften kommen, die direkt über den Google Server abrufbar sind
  • Eine sehr stark Browser-unabhängige Schreibweise für die @font-face Regel is 
    @font-face { font-family: 'Whatever'; /* Erstellen neue Schriftart Whatever */
	src: url('myfont.eot'); /* IE9 Compatibility Modus - Liefert Support für IE ab Version 4*/
	src: url('myfont.eot?iefix') format('eot') /*IE6-IE8*/, url('myfont.woff') format('woff') /*Modern*/,
		 url('myfont.ttf') format('truetype'), /*Mobile*/ url('myfont.svg') format('svg'); /*Fallback*/ }
