Zusätzliche Effekte

  • jQuery UI erweitert auch die Palette der Effekte, die man bisher z.B. in folgendem Code verwendet hat, 
    $('#mydiv').animate({ /* Zu animierende Eigenschaften */} , {
	duration: 2000, //Zeit in ms
	complete: function() { }, //Was passiert nach Abschluss?!
	easing: 'linear' }); //Genau hier legen wir das Easing fest - in diesem Fall linear
  • Zusätzlich kann man über specialeasing noch Effekte zu jeder Eigenschaft festlegen
