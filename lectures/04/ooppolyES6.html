Polymorphie in JavaScript mit ES 6

Polymorphie in ES 6 funktioniert mittels des Schlüsselwortes extends

     class Parent {
      constructor(Name){
        this.LastName = Name;
      }
      speak(s){
        console.log("I glab des is a so: "+s);
      }
    }

    class Child extends Parent{
      constructor(LastName, _FirstName){
        super(LastName);
        this.FirstName = _FirstName;
      }
      speak(s){
        console.log("Meine Eltern hätten gesagt: "); console.log(super.speak(s));
        console.log("Aber ich sage:"); console.log("Ich weiß, es ist derart: "+s);
      }
    }
  • Wir können das super Schlüsselwort verwenden um auf die Funktionen der Eltern-Klasse zuzugreifen.
  • Wir müssen das super Schlüsselwort verwenden um den Konstruktor der Elternklasse aufzurufen befor wir Code im Kind-Konstruktor schreiben können.
