"Klassen" in JavaScript vor ES 6

  • In JavaScript gab es vor ES 6 kein Schlüsselwort um Klassen direkt zu erzeugen (Tricksen ist jedoch erlaubt!)
  • Jedoch bietet uns JS drei Möglichkeiten eine Klasse zu erstellen
    • Als Objekt (hier haben wir aber keinen Konstruktor und daher nur eine Instanz verwenden) über 
      var NameDerKlasse = {};//Erstellen des Object-Literals
NameDerKlasse.WertDerEigenschaft = 5;//Eigenschaft
NameDerKlasse.FunktionsName = function() {...};//Methode
    • Als anonyme Funktion mit dem new Operator (hier haben wir einen sinnlosen Konstruktor) 
      var NameDerKlasse = new function(...) { //Erstellen und zuweisen der anonymen Methode
	this.WertDerEigenschaft = 5;//Eigenschaft
	this.FunktionsName = function() {...}; };//Methode
    • Als benannte Funktion mit dem new Operator (hier haben wir einen sinnvollen Konstruktor) 
      function UnsereKlasse(...) { //Erstellen der benannten Methode
	this.WertDerEigenschaft = 5;//Eigenschaft
	this.FunktionsName = function() {...}; };//Methode
var NameDerKlasse = new UnsereKlasse(...);//erstellt Instanz!
