AJAX mit jQuery

  • Alle AJAX Abfragen laufen in jQuery über $.ajax(), auch speziellere Methoden wie $.getJSON()
  • jQuery bietet uns standardmäßig Ereignisse und Möglichkeiten die Abfrage individuell zu gestalten
  • Die grundsätzliche Syntax lautet 
    var settings = {}; //Einstellungen für die Abfrage
var ajaxObjekt = $.ajax('url der abfrage', settings); //Abfrage tätigen
ajaxObjekt.abort(); //Abfrage abbrechen, optional wenn notwendig
    die zurückgegebene Variable ist optional - mit ihr ist es z.B. möglich den Vorgang abzubrechen
  • Fast alle Möglichkeiten stecken in dem übergebenen settings-Objekt, z.B. 
    var settings = {
	async: true, // könnten den Request auch Blockend aufbauen - mit false
	cache: false, // jQuery sendet in der URL ein _=[TIMESTAMP] mit um Caching zu verhindern
	complete: function(obj, status) { }, // Callback wenn abgeschlossen
	crossDomain: false, // aus Sicherheitsgründen notwendig - true wenn Zugriff auf anderen Server
	data: { }, // Objekt das an den Server mittels Name-Wert-Paaren geschickt wird (a la HTML-Form)
	dataType: 'xml', // Datentyp des AJAX-Requests - auch JSON, Script bzw. HTML sind möglich
	error: function(obj, status, err) { }, // Callback wenn Fehler auftritt
	success: function(data, status, obj) { }, // Callback wenn erfolgreich abgeschlossen
	type: 'GET' } // bestimmt den Typ des Requests - auch POST, PUT etc. sind möglich
