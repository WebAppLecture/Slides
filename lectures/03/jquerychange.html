Elemente ändern

  • Einer der Vorteile im Verwenden von jQuery liegt in der (universalen) Syntax zum Ändern von Elementen
  • Im Prinzip gelten folgende Regeln:
    • Will man das Attribut eines HTML Tags ändern (oder auslesen) so verwendet man attr(), z.B. 
      var getter = $('#beispielID').attr('href');//Speichert den Wert des Attributes href in getter
$('#beispielID').attr('href', 'beispiel.html');//Setzt den Wert (und gibt jQuery-Instanz zurück)
    • Will man den Style eines HTML Elements ändern (oder auslesen) so verwendet man css(), z.B. 
      var getter = $('#beispielID').css('display');//Speichert den Wert der Eigenschaft display des Styles
$('#beispielID').css({ 'display' : 'none'});//So kann man auch mehrere Werte setzen - Objekt!
    • Will man das Ereignis eines HTML Elements ändern (oder auslösen) so verwendet man bind(), z.B. 
      $('#beispielID').bind('click');//Führt das Click-Ereignis aus
$('#beispielID').click();//Sehr viel kürzere Alternative - speziell für Click
$('#beispielID').bind('click', function() {});//Setzt das Click-Ereignis
$('#beispielID').click(function() {});//Ebenfalls kürzere Alternative
