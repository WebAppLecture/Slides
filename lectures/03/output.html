Das output Element

  • Semantischer Tag um als Platzhalter für die Ausgabe von Skripts zu dienen
  • Besitzt keine Eigenschaft value
  • Ist nur für die clientseitige Darstellung relevant (wird nicht übertragen)
  • Beim Reset des Formulars wird auch der Inhalt des output Elements zurückgesetzt
  • v.a. sinnvoll für Berechnungen, temp. Anzeigen und zur Kontrolle von Benutzereingaben
  • Ein einfaches Beispiel: 
    <form onsubmit="return false" oninput="o.value = a.value*1 + b.value*1">
  <input name="a" id="a" type="number" step="any"> +
  <input name="b" id="b" type="number" step="any"> =
  <output name="o" for="a b"></output>
</form>
