Iteratoren (ES 6)

  • Iteratoren erlauben und durch eigene Klassen mittels for( let x of y) zu laufen.
  • Ein Iterator gibt immer ein Objekt mit einer next() function zurück.
  • next() gibt immer ein Paar mit einmal dem aktuellen value und der Bool-Variable done zurück, die das Ende anzeigt.
        class MyDiary{
          constructor(){
            this.entries = ["Dear", "Diary", "today", "was", "a" , "great", "day"];
          }
          [Symbol.iterator](){
            let i = 0;
            const messages = this.entries;
            return {
                next() { //Kurzschreibweise für next : function() { /* ... */ }
                  if(i < messages.length)
                    return {value: messages[i++], done: false};
                  else
                      return {value: undefined, done: true};
                  } //end next function
            }; } };
        let d = new MyDiary();
        for(let entry of d) console.log(entry);
