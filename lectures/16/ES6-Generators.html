Generatoren (ES 6)

  • Generatoren sind Funktionen, die uns einen Iterator zurückgeben, den wir aufrufen können
  • Eine sehr angenehme Art Generatoren zu schreiben geht mithilfe des yield Schlüsselwortes
    •         function* myOwnGenerator(){ //der Stern ist wichtig! Markiert Funktion als Generator
          yield "Dear";
          yield "Diary";
          yield "today";
          yield "was"; yield "a"; yield "great"; yield "day";
          return ; //beendet den Generator;
        }
        //Hole den Iterator
        let myIterator = myOwnGenerator();
        //Durchgehen:
        console.log(myIterator.next()); // { value: "Dear", done: false}
        console.log(myIterator.next()); // { value: "Diary", done: false}
        // ....
        //Alternativ
        for(let entry of myOwnGenerator())
          console.log(entry);
  • Es sind auch kompliziertere Interaktionen möglich (.next() lässt sich mit Übergabeparameter aufrufen1)
