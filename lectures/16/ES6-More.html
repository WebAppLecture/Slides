Noch mehr ES 6 Features (unvollständig1)

  • Map: Optimiertes Datenobjekt für Schlüssel-Wert-Paare.
    Wichigste Methoden .get(), .has(),.set(), .clear() und .delete()
  • Set: Optimiertes Datenobjekt für Sammlungen eineindeutiger Werte (keine Dopplungen)
    Wichigste Methoden .add() und .delete() sowie die Eigenschaft size
  • Weak Maps und Weak Sets: Minimalisierte Versionen von Map und Set für noch speziellere Anforderungen.
  • Template strings: 
            let Money = 100;
        console.log(`Der Mantel kostet ${Money} €`); //MUSS mit diesem Akzent ` geschrieben werden
  • Symbol um einzigartige Identifizierer zu erstellen 
                const Token1 = Symbol(); const Token2 = Symbol();
            Token1 === Token2; //false!
  • Reflection: Ein Objekt kennt seine eigenen Schlüsselwörter 
             const obj = { a: 4, b: "hi", c : x => 2*x};
         Reflect.ownKeys(obj); // [a,b,c]
