Neue Grundelemente

  • HTML5 führt einige neue Elemente ein, welche direkt nichts anderes als ein div darstellen
  • Der Vorteile in der Benutzung dieser neuen Tags liegen auf der Hand:
    • Bessere Auslesbarkeit durch Screenreader
    • Strukturierterer Aufbau
    • Direkteres und damit auch leichteres Styling
  • Die Abwärtskompabilität ist bis auf den IE sehr gut gegeben1
Der HTML Header Tag im Detail (für Titel) Der HTML Footer Tag im Detail (für Fußzeilen) Der HTML Section Tag im Detail (für Sektionen) Der HTML Article Tag im Detail (für Artikel) Der HTML Nav Tag im Detail (für Navigation) Der HTML Aside Tag im Detail (für Randnotizen)

1Für ältere IEs gibt es jedoch über JavaScript einen sehr leichten Trick (später)
